# 读书笔记
> 建立分支来同步个人读书进度，互相监督，互相做code review

## 目录格式
```
├── README.md
└── golang （用来区分项目，例如go，数据库等等）
    ├── README.md （目录）
    ├── book （书籍类）
    │   ├── README.md
    │   └── build-web-application-with-golang
    │       ├── README.md （项目介绍）
    │       ├── content.md （笔记）
    │       └── src （示例代码）
    │           └── demo_http.go
    └── source （源码类）
        ├── README.md
        └── httprouter
            ├── README.md
            └── content.md
```

可以从书籍，博客，源码等等来整理
